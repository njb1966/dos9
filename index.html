Simple. Visible. Yours.

DOS/9 is a native text operating system for people who like systems they can understand: a shell-first kernel, visible filesystems, and small-web tools that feel practical instead of performative.

DOS/9 on a calm paper background. Green accents, steel text, no rogue aesthetic.

A practical small-web OS, not a retro costume.

DOS/9 leans toward the fediverse and small-web crowd because the software is meant to be read, trusted, and used daily. The visual style follows the same rule: restrained, legible, and a little warm.

The shell is the center

Users drive the system through text, not through a hidden launcher or a simulated desktop layer.

Files everywhere

Devices, processes, modules, and network state are exposed through filesystem paths and normal tools.

Readable by design

The goal is not maximal abstraction. The goal is a system a motivated person can inspect and extend.

What exists now

The project already boots, reaches a shell, runs userland programs, and includes real networking clients for the small web.

Current milestone
